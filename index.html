TECHNOLOGIES

  • Youtube API

    Youtube Data API v3를 이용해 필요한 영상 정보를 가져왔고, Youtube Player API를 이용해 동영상을 재생시켰습니다.

  • Node js + Maria DB

    Node js를 통해 클라이언트와 비동기 통신을 수행하였고, 클라이언트 정보를 공유하기 위해 Maria DB에 저장하였습니다.

  • Firebase

    Firebase Admin, Firebase Cloud Messaging 등을 사용하였습니다.

  • Retrofit2 + Gson

    Http통신을 위해 사용했습니다.

  • Glide + Palette

    Youtube 서버에서 받은 이미지 url을 Glide로 로드하였고 Palette로 색을 추출하였습니다.

  • Event Bus

    Firebase 서버에서 받은 메세지를 Event Bus를 통해 전송하였습니다.
